# MyWatchlist - Backend API

A RESTful backend service built with Java and Spring Boot for managing and tracking personal media consumption. This API serves as the core backend for MyWatchlist, aggregating disparate data from external sources into a unified platform. 

## Core Technologies
* **Framework:** Java, Spring Boot
* **Security:** Spring Security, JWT (JSON Web Tokens)
* **Persistence:** Spring Data JPA, Relational Database (PostgreSQL/MySQL/SQL Server)
* **Integrations:** TMDB API (Movies/TV Shows), Jikan API (Anime)
* **Build Tool:** Maven
